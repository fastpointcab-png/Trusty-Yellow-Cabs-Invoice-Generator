# Taxi Invoice Generator (Responsive with PDF download)

Features:
- Responsive layout with tabs (Form / Preview) optimized for mobile and desktop.
- Yellow-black Trusty Yellow Cabs theme.
- Client-side PDF download (filename fixed as `Invoice.pdf`).
- Node.js + Express backend to save invoice JSON in `invoices/`.

## Quick start
1. unzip and `cd taxi-invoice-project-responsive`
2. `npm install`
3. `node server.js`
4. Open `http://localhost:3000`
